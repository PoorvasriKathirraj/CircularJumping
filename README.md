CircularJumping

This project solves a circular chair jumping problem using an efficient algorithm.
A group of people are seated in a circle, and each position allows movement left or right by a fixed number of steps.
The goal is to determine the minimum number of jumps required to reach a target chair from a starting chair.
The solution uses Breadth First Search (BFS) to guarantee the shortest path.
It also handles circular movement using modular arithmetic.
If the destination cannot be reached, the program correctly returns -1.
